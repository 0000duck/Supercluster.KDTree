BoundedPriorityListTElement, TPriorityAdd Method
Attempts to add the provided item. If the list is currently at maximum capacity and the elements priority is greater than or equal to the hiest priority, the item is not inserted. If the item is eligable for insertion, the upon insertion the item that previously had the largest priority is removed from the list. This is an O(log n) operation.
