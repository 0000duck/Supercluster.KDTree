BoundedPriorityListTElement, TPriority
A list of limited length that remains sorted by TPriority. Useful for keeping track of items in nearest neighbor searches. Insert is O(log n). Retrieval is O(1)
KDTreeTDimension, TNode
Represents a KD-Tree. KD-Trees are used for fast spatial searches. Searching in a balanced KD-Tree is O(log n) where linear search is O(n). Points in the KD-Tree are equi-length arrays of type TDimension. The node objects associated with the points is an array of type TNode.
