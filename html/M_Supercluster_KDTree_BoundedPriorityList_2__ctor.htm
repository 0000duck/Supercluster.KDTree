Initializes a new instance of the BoundedPriorityListTElement, TPriority class. Note: You should not have allocate set to true, and the capacity set to a very large number. Especially if you will be creating and destroying many BoundedPriorityListTElement, TPriority very rapidly. If you ignore this advice you will create lots of memory pressure. If you don't understand why this is a problem you should understand the garbadge collector. Please read: https://msdn.microsoft.com/en-us/library/ee787088.aspx
